# FuckX
A Chrome extension to declutter X by hiding ads, premium prompts, metrics, algorithmic feeds, and restoring the classic bird logo.

## Installation
1. Download and extract this repository to your computer.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Toggle on **Developer mode** in the top right corner.
4. Click **Load unpacked** in the top left.
5. Select the extracted folder containing the extension files.
