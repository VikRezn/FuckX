# FuckX
A Chrome extension to declutter X by removing unwanted features, algorithmic feeds, and upselling prompts.

## Features
* **Ad & Promo Blocker:** Hides timeline ads and promotional banners.
* **Navigation Cleanup:** Removes Premium, Creator Studio, and Grok links from the main sidebar.
* **Feed Customization:** Hides "Who to follow," "Today's News," "What's happening," "Relevant people," "You might like," and "Live on X" sections.
* **Engagement Control:** Mutes verified Blue Check badges and hides all tweet metrics (replies, retweets, likes, views).
* **Classic Logo Restoration:** Replaces the X logo with the classic Twitter Bird logo on the page and in the browser tab, with options to toggle between white and blue variants.

## Installation
1. Download and extract this repository to your computer.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Toggle on **Developer mode** in the top right corner.
4. Click **Load unpacked** in the top left.
5. Select the extracted folder containing the extension files.
